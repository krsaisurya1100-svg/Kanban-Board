
# Kanban Board

Built a simple Kanban board to manage tasks and understand how tools like Trello work under the hood. This project focuses on UI structure, state handling, and implementing drag-and-drop functionality.

## What it does

* Tasks are organized into 3 stages:

  * Not Started
  * In Progress
  * Completed
* Add tasks to any column
* Move tasks across columns using drag and drop
* Instant visual updates when tasks are moved

## Features

* Column-based task management
* Drag and drop between sections
* Simple and clean UI
* Lightweight and easy to use

## Tech stack

* HTML
* CSS
* JavaScript

## How to run

Clone the repo and open locally:

```bash
git clone https://github.com/your-username/kanban-board.git
cd kanban-board
```

Then open `index.html` in your browser.
